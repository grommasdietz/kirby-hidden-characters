# Contributing

Thanks for your interest in contributing to this plugin.

---

## Local setup

- [ ] Install [Composer dependencies](docs/contributions/setup.md#composer)
- [ ] Install [Node dependencies](docs/contributions/setup.md#node)

---

## Before you open a PR

- [ ] Run [tests](docs/contributions/tests.md)
- [ ] Use [Conventional Commit messages](.gitmessage)

---

## What we accept

- [ ] Simplify or remove unnecessary complexity
- [ ] Update outdated or deprecated code
- [ ] Reduce duplication and improve maintainability
- [ ] Align the plugin with Kirby's recommended practices
- [ ] Add new, well-documented features

---

## Documentation

- [ ] Follow the contributor documentation hub: [docs/contributions](docs/contributions/index.md)
